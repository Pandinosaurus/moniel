Moniel

Interactive Notation for Computational Graphs.

Notation for Graph Construction

Moniel was conceived as a natural notation for constructing graphs. Visual similarity to actual graphs is not coincidental.

Smart Acronym Recognition

Using fewer keystrokes means faster writing. But more importantly, shorter acronyms also means faster reading.

Automatic Graph Layout

No more unreadable spaghetti diagrams. No more hand-positioning every node. Perfect layout everytime.

Instant Visual Feedback

Visualization is always up-to-date, so you don't have to guess if you got it right or wrong.
